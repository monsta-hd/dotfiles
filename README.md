# dotfiles
## How to install
Get the repository
```bash
git clone git@github.com:monsta-hd/dotfiles.git
cd dotfiles/
```
## How to run
* Fill `password.txt`
* Install all dotfiles by running
```bash
make
```
&nbsp; &nbsp; **or** install specific dotfiles by running corresponding makefile, e.g.
```bash
cd alias/ && make && cd ..
```
